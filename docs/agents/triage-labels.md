# 分流标签

技能体系使用五种规范分流角色。本文件将这些角色映射到本仓库 Issue 跟踪器中的实际标签字符串。

| mattpocock/skills 中的标签 | 本仓库标签 | 含义 |
| -------------------------- | ---------- | ---- |
| `needs-triage`             | `needs-triage` | 维护者需要评估此 Issue |
| `needs-info`               | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent`          | `ready-for-agent` | 规格完整，可供 AFK Agent 实现 |
| `ready-for-human`          | `ready-for-human` | 需要人工实现 |
| `wontfix`                  | `wontfix` | 不予处理 |

当技能提到某分流角色（例如「应用 AFK-ready 分流标签」）时，使用上表对应的标签字符串。

若实际使用的词汇不同，请修改右列以保持一致。
