# mini-agent
Building a ai agent in go from scratch. 

## Setup
I use ollama to run llm.

- First, download a model.
```sh
ollama pull <your_model_name>
```
- Second, list installed model.
```sh
ollama list
```
- Finally, run the model.
```sh
ollama run <your_model_name>
```
More details: https://docs.ollama.com/cli
